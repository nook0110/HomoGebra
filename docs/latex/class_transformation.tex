Construct transformation from movement of 4 points. 

This transformation will do homography (preimage-\/\texorpdfstring{$>$}{>}image) A projective transformation of the plane is defined by specifying four pairs of corresponding mapping points. For correct homography three points of the four images or preimages should not lie on the same line. (Otherwise it will degenerate)
